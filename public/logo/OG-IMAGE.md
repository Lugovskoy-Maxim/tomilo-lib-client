# Превью ссылок в соцсетях (og:image)

Чтобы в Telegram, VK, Facebook и др. отображалась картинка при отправке ссылки на сайт или тайтл, нужен **растровый** формат (PNG или JPEG). SVG не поддерживается.

## Дефолтная картинка

Положите в эту папку файл **og-default.png** (рекомендуемый размер 1200×630 px).

Он будет использоваться:
- для главной страницы;
- для страниц тайтлов без обложки.

Либо задайте свой URL в `.env`:
```
NEXT_PUBLIC_OG_DEFAULT_IMAGE=/logo/og-default.png
```

## Обложки тайтлов

Если обложки отдаются с другого хоста (например с API), укажите в `.env`:
```
NEXT_PUBLIC_IMAGE_URL=https://ваш-api-домен.ru
```
Тогда в мета-теги og:image попадёт абсолютный URL обложки с правильного хоста.
