# Превью ссылок в соцсетях (og:image)

Чтобы в Telegram, VK, Facebook и др. отображалась картинка при отправке ссылки на сайт или тайтл, нужен **растровый** формат (PNG или JPEG). SVG не поддерживается.

## Дефолтная картинка (обязательно)

**Файл `og-default.png` должен лежать в этой папке** — иначе превью для главной и тайтлов без обложки будет пустым или с ошибкой.

- Рекомендуемый размер: **1200×630 px** (PNG или JPEG).
- Используется для главной страницы и для страниц тайтлов без обложки (или с обложкой в формате SVG; SVG в превью подменяется на эту картинку).

Либо задайте свой абсолютный URL в `.env`:
```
NEXT_PUBLIC_OG_DEFAULT_IMAGE=/logo/og-default.png
```

## Обложки тайтлов

Если обложки отдаются с другого хоста (например с API), укажите в `.env`:
```
NEXT_PUBLIC_IMAGE_URL=https://ваш-api-домен.ru
```
Тогда в мета-теги og:image попадёт абсолютный URL обложки с правильного хоста. Обложки в формате SVG автоматически заменяются на дефолтную картинку.
